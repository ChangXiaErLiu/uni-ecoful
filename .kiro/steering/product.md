# Product Overview

**粤风环保AI系统** (Yue Feng Environmental AI System) is a comprehensive environmental protection AI platform designed for the Chinese market. The system provides intelligent tools for environmental compliance, report generation, and knowledge management.

## Core Features

- **AI Chat Assistant**: Environmental domain Q&A with intelligent responses
- **Report Generation**: Template-based intelligent document creation for environmental reports
- **Compliance Review**: Automated document compliance checking
- **Knowledge Base**: Document management and search capabilities
- **Acceptance Reports**: Project acceptance report generation (Beta)
- **EIA Reports**: Environmental Impact Assessment report assistance (Beta)
- **Field Survey**: On-site inspection guidance generation (Beta)

## Target Platforms

- WeChat Mini Program (primary)
- H5 Web Application
- Mobile Apps (Android/iOS via uni-app)

## User Interface

The application uses a card-based interface with a clean, modern design focused on environmental green themes. Navigation follows standard mobile patterns with tab bars on mobile and sidebar navigation on desktop.
